# Portfolio

A modern personal portfolio website built with React and Vite.

## Features

- **Modern Stack**: Built with React 19 and Vite for fast development and optimal performance
- **Responsive Design**: Mobile-first approach ensuring great experience across all devices
- **Fast Loading**: Optimized build with Vite's lightning-fast bundling
- **Clean Code**: ESLint configuration for code quality and consistency

## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone https://github.com/riosjmo/portfolio.git
cd portfolio
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint

## Deployment

The site can be easily deployed to platforms like:

- Vercel
- Netlify
- GitHub Pages
- Any static hosting service

To build for production:

```bash
npm run build
```

The `dist` folder will contain the optimized production build.

## Tech Stack

- **React 19** - UI library
- **Vite** - Build tool and development server
- **ESLint** - Code linting
- **CSS3** - Styling
